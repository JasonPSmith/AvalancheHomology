# Avalanche Homology

A Python library for computing the avalanche homology of (di)graphs.

See [Avalanche homology of digraphs via sandpile dynamics]() for the mathematical background.

# Installation

## Clone the repository

```bash
git clone https://github.com/JasonPSmith/AvalancheHomology.git
cd AvalancheHomology
````

## Install in editable mode

```bash
pip install -e .
```
